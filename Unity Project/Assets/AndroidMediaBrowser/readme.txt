Android Media Browser (v 1.1.1)
http://diversido-mobile.com/


Overview
--------

Android Media Browser provides a Unity developer friendly way to access media content on android devices. Audio, Video and Image media types supported. You can either invoke native media browsing tool and get a callback with a user media selection, or use media search routines to implement your own UI for media browsing.


Manifest Tool
-------------

Android Media Browser comes with manifest tool, use "Diversido" -> "Android Media Browser Setup" command from menu. From this setup window you can create default Android manifest file, if you don't have one. Then you can add picker activities and register application as a viewer for different media types.
